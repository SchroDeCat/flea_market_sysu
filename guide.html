部署教程

项目Django2.0.4及Python3.5开发，部署在Ubuntu服务器上，部署效果可参考sysufleastore.com

环境说明

python3.5及以上

Django 2.0.4

SQLite3

Ubuntu 16.04 AWS版本

1.配置服务器

项目以在AWS的EC2部署为例

Nginx + uwsgi + django的交互逻辑为： web client<==>Nginx<==>socket<==>uwsgi<==>django

配置Python & pip环境

更新源和包
sudo apt-get update
sudo apt-get upgrade

AWS默认为python3，没有安装pip
ln -s /usr/bin/python3 /usr/bin/python
sudo apt-get install python3-pip
ln -s /usr/bin/pip3 /usr/bin/pip

配置安全组

在AWS安全组中设置入站、出站允许所有流量、所有端口所有来源

配置防火墙

检查ufw, iptables,firewalld等防火墙是否禁用指定端口

配置uwsgi

pip install uwsgi

在/etc/nginx/sites-available/目录下，新建flea_market.conf：

# flea_market.conf

# the upstream component nginx needs to connect to
upstream django {
    # server unix:///path/to/your/mysite/mysite.sock; # for a file socket
    server 127.0.0.1:8001; # for a web port socket (we'll use this first)
}

# configuration of the server
server {
    # the port your site will be served on
    listen      80;
    # the domain name it will serve for
    server_name example.com; # substitute your machine's IP address or FQDN
    charset     utf-8;

    # max upload size
    client_max_body_size 75M;   # adjust to taste

    # Django media
    location /media  {
        alias /path/to/your/mysite/media;  # your Django project's media files - amend as required
    }

    location /static {
        alias /path/to/your/mysite/static; # your Django project's static files - amend as required
    }

    # Finally, send all non-media requests to the Django server.
    location / {
        uwsgi_pass  django;
        include     /path/to/your/mysite/uwsgi_params; # the uwsgi_params file you installed
    }
}

将以上文件软连接到/etc/nginx/sited-enabled

sudo ln -s /etc/nginx/sited-available/flea_market.conf /etc/nginx/sites-enabled/

2. 配置项目

从github上拷贝项目

git clone https://github.com/SchroDeCat/flea_market_sysu.git
cd flea_market_sysu
pip install -r requirements.txt

为了运行Nginx，需要手机Django的static文件，首先需要在项目setting.py文件中增加：


STATIC_ROOT = os.path.join(BASE_DIR, "static/")

然后运行：

python manage.py collectstatic

重启nginx:

sudo /etc/init.d/nginx restart

使用uwsgi在80端口运行项目

uwsgi --socket 127.0.0.1:3031 --socket 127.0.0.1:8001 --chdir /home/ubuntu/flea_market_sysu/ --wsgi-file flea_market/wsgi.py --master --processes 1 --threads 1 --stats 127.0.0.1:9191
